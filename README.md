# WebJep!
A web application version of my original PyQt [Jep!](https://github.com/sombreroman55/jep)
application. This project runs in your browser and is primarily designed for connecting with friends
and playing online using WebRTC, though a local version for just presenting the clues with everyone
in the same room can also be run.

This repository contains both the server and client code for the application. The server is written
in TypeScript and uses [Express](https://expressjs.com/) and [Socket.IO](https://socket.io/) to
serve the client code and handle the WebRTC connections. The client is written in TypeScript and
Vue3 using [Vite](https://vitejs.dev/) as the build tool.
